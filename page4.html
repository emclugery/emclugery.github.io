Projet 4 : Comparaison d'approches algorithmiques


Dans ce projet de comparaison d'approches algorithmiques pour le jeu Puissance 4, l'objectif est d'analyser et de comparer différentes méthodes de résolution du jeu afin de déterminer leur efficacité et leur performance. En équipe de deux, notre mission pour ce projet est d'améliorer le programme de Puissance4 afin de permettre à l'utilisateur de jouer contre l'ordinateur. Ce projet s'est déroulé en plusieurs parties, le détails des stratégies utilisées pour le programme, le codage de ces stratégies et enfin l'évaluation qui consite à faire jouer les stratégies et évaluer leur performance.


1ère étape :
Présentation des stratégies :


Dans cette première partie, nous avons conçu deux stratégies permettant de trouver la meilleur colonne où placer un pion et rédigé en détail leurs fonctionnements


2ème étape :
Code Source :


Un code en langage C nous a été fournis pour que l'on puisse le tester avec les stratégies développées. Ci-dessous se trouve le code du Puissance 4 anisi que les deux stratégies implémentées en C.


3ième étape :
Puissance 4 amélioré :


La dernière partie du projet consiste à évaluer la performance de chaque stratégie et comparer leur efficacité afin de trouver la meilleure. Pour cela, un dossier nommé Evaluation nous a été fourni pour tester les performances. Grâce au dossier, il nous à juste fallut placer les stratégies dans le code pour qu'il nous indique la plus efficace en terme de coups. Une fois la meilleure stratégie choisie, un concours à été organisé pour savoir quelle stratégie est la plus performante
